# Elevate Labs Cyber Security Internship

## Task 5 – Capture and Analyze Network Traffic Using Wireshark

### Objective
Capture live network packets using Wireshark and identify different network protocols.

### Tool Used
- Wireshark

### Procedure
1. Installed Wireshark.
2. Started packet capture on the active Wi-Fi interface.
3. Generated network traffic by browsing websites.
4. Stopped packet capture after approximately one minute.
5. Applied protocol filters:
   - DNS
   - TCP
   - TLS
6. Analyzed captured packets.
7. Saved the packet capture as `network_capture.pcapng`.
8. Documented the observations.

### Protocols Identified
- DNS
- TCP
- TLS

### Files Included
- network_capture.pcapng
- Network Traffic Analysis Report.txt
- Screenshots
